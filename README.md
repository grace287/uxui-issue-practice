채용관리 서비스입니다. <br><br>

Q. 기업들이 채용과정에서 어떤 어려움을 겪고 있을까요?
  <li>매년 바뀌는 채용절차 법률</li>
  <li>다양한 경로의 지원자 관리</li>
  <li>합격 / 불합격 통보 누락</li> <br>
  <br>
